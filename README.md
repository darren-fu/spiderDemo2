# node-webcrawler

### 使用方法
```shell
node index.js
```
按照控制台的提示进行操作即可
http://127.0.0.1:3000/

### 特性
- 添加配置进行界面化，不需要手动更改代码
- 类似jQuery的选择器易于书写
